# Round Robin Scheduling Algorithm

A Python implementation of the **Round Robin CPU Scheduling Algorithm** that schedules processes using a user-defined time quantum. The program calculates key scheduling metrics and visualizes the results using graphs.

## Features

- Implements Round Robin Scheduling
- Calculates CT, TAT, WT, RT, and Throughput
- Generates a Gantt Chart
- Displays Waiting Time and Turnaround Time graphs
- Compares performance for different time quantum values

## Tech Stack

- Python
- Matplotlib
- PyCharm

## How It Works

1. Enter the number of processes.
2. Provide the Process ID, Arrival Time, and Burst Time for each process.
3. Enter the desired Time Quantum.
4. The program executes the Round Robin Scheduling algorithm.
5. It displays the scheduling table with all performance metrics.
6. Graphs are generated to visualize execution and compare performance across different time quantum values.
